## 🚀 วิธีติดตั้ง (ทำครั้งเดียว)

### ขั้นตอนที่ 1 — ตั้งค่า Google Apps Script

1. ไปที่ [script.google.com](https://script.google.com) → **New Project**
2. ลบโค้ดเดิมออก → วางเนื้อหาจากไฟล์ `Code.gs` ทั้งหมด
3. บันทึก (Ctrl+S)
4. คลิก **Deploy** → **New deployment**
5. ตั้งค่า:
   - Type: **Web App**
   - Execute as: **Me**
   - Who has access: **Anyone**
6. คลิก **Deploy** → อนุมัติสิทธิ์ถ้าระบบถาม
7. **Copy URL** ที่ได้ (จะมีรูปแบบ `https://script.google.com/macros/s/XXXX/exec`)

### ขั้นตอนที่ 2 — แก้ URL ใน index.html

เปิด `index.html` ค้นหาบรรทัดนี้:
```javascript
const SCRIPT_URL = 'https://script.google.com/macros/s/REPLACE_WITH_YOUR_SCRIPT_ID/exec';
```
แทนที่ด้วย URL ที่ได้จากขั้นตอนที่ 1

### ขั้นตอนที่ 3 — อัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่บน GitHub (ตั้งชื่อตามต้องการ)
2. อัปโหลดไฟล์ `index.html`
3. ไปที่ Settings → Pages → Source: **Deploy from branch** → Branch: **main**
4. รอสักครู่ → เว็บไซต์จะพร้อมใช้งานที่ `https://username.github.io/repo-name/`

## 📱 การใช้งาน

### สำหรับพนักงานหน้างาน (บันทึกข้อมูล)
1. เปิดเว็บจากมือถือ
2. กรอกวันที่ + เลือกชื่อผู้ตรวจสอบ
3. เลือกระดับความเสี่ยงที่ต้องการตรวจ (High / Medium / Low)
4. กรอกสถานะและติ๊กรายการตรวจสอบให้ครบทุกเครื่อง
5. กดบันทึก → ข้อมูลจะเข้า Google Sheet ทันที

### สำหรับหัวหน้าหน่วยงาน (Dashboard)
1. เปิดเว็บ → คลิกแท็บ **📊 Dashboard**
2. ดูสรุปสถิติ: รายการทั้งหมด / พร้อมใช้ / มีปัญหา / ยกเลิก
3. กรองตามระดับความเสี่ยง: ทั้งหมด / High / Medium / Low
4. ดูตารางรายการล่าสุด 50 รายการ
5. ดูสรุปสถานะแต่ละเครื่อง (เครื่องที่ 1 และ 2)
6. กด **🔄 รีเฟรช** เพื่ออัปเดตข้อมูล Realtime

## 🎨 ระดับสี (Color Code)

| สี | ความหมาย |
|----|----------|
| 🔴 แดง (High Risk) | เครื่องช่วยหายใจ, เครื่องกระตุกหัวใจ, Auto CPR, AED |
| 🟡 เหลือง (Medium Risk) | IV Pump, Suction, NIBP, EKG, Ultrasound, Oximeter, จี้ไฟฟ้า |
| 🟢 เขียว (Low Risk) | วัดความดัน, ที่ส่องหู, ชั่งน้ำหนัก, ปรอทวัดไข้, หูฟัง |

| สถานะ | ความหมาย |
|-------|----------|
| ✅ พร้อมใช้/มี | เครื่องพร้อมใช้งานเต็มที่ |
| ⚠️ มีปัญหา แต่ยังใช้อยู่ | ใช้งานได้แต่มีข้อบกพร่อง ต้องติดตาม |
| ❌ ยกเลิกการใช้งาน | เครื่องไม่สามารถใช้งานได้ |
